# Breast Cancer Prediction

## Description

This project predicts whether a breast tumor is malignant or benign using the Logistic Regression algorithm.

The dataset is preprocessed, normalized using MinMaxScaler, and then used to train a machine learning model.

## Features

- Breast cancer prediction
- Data preprocessing
- Feature normalization
- Logistic Regression model
- Model evaluation
- Model serialization using Joblib

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib

## Requirements

- Pandas
- NumPy
- Scikit-learn
- Joblib

## Installation

pip install -r requirements.txt
## Run

Run the notebook (.ipynb) using Jupyter Notebook or JupyterLab.

## Dataset

Breast Cancer Wisconsin Diagnostic Dataset

## Output

- Model Accuracy
- Number of Wrong Predictions
- Saved trained model (breast_cancer_model.pkl)
- Saved scaler (scaler.pkl)
