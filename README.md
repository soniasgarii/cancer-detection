# Face Recognition System

## Description

A computer vision project for detecting faces and eyes in real-time using OpenCV and Haar Cascade classifiers.

## Features

- Real-time face detection
- Eye detection inside detected faces
- Live webcam processing
- Automatic handling when no face is detected

## Technologies

- Python
- OpenCV
- Haar Cascade

## Requirements

- OpenCV

## Installation

pip install -r requirements.txt

## Run

Run the notebook (.ipynb) using Jupyter Notebook or JupyterLab.
